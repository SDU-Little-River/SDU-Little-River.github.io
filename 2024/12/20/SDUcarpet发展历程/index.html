SDU-carpet-addition发展历程

发表于2024-12-20|更新于2024-12-21
|总字数:827|阅读时长:2分钟

2024

01-30

为了方便首届Minecraft数学建模比赛的开展，梦空制作了最初版的SDUcraft-carpet-addition，增加规则skipCropLightCheck
启用后，作物将跳过种植和生长时的光照检测。

04-30

SDU_V想在三周目生存服中制作潜影贝农场，为了修复1.20.4潜影贝的bug，我为SDUcraft-carpet-addition添加了规则armorStandIgnoreShulkerDamage 启用后，潜影贝的导弹将不会摧毁盔甲架

05-02

在梦空的帮助下，从carpet-AMS-addition中移植了易碎深板岩规则

08-20

新增规则itempickupDelay用于控制被玩家丢出的物品再次被拾取的冷却

09-27

为了加速施工，服务器经常处于加速状态，有许多新玩家常常反映不知道如何调整游戏速度。加入规则tickRateChangedMessage当新玩家加入游戏时，若目前游戏速度不为20gt/s,自动将恢复正常游戏速度的指令发送给玩家。同时，在游戏速度改变时，允许玩家使用/leavemessage 指令进行留言

10-9

伪和平开关太远没人去，反正召唤假人开关伪和平已经够notvanilla了，那就更notvanilla一点吧（，加入简易伪和平系统，允许玩家通过/fakepeace指令快速操控伪和平

10-21

新增简易命令系统，支持快速操控伪和平，镜像服开关等等

10-21

新增简易命令系统，支持快速操控伪和平，镜像服开关等等

2024

01-30

skipCropLightCheck

使作物跳过光照检测

  • 类型：boolean
  • 默认值：false
  • 有效值：true, false

04-30

armorStandIgnoreShulkerDamage

阻止潜影贝导弹摧毁盔甲架

  • 类型：boolean
  • 默认值：false
  • 有效值：true, false

05-02

brittleDeepSlate

将深板岩的硬度变为1.6刚好可以被急迫2效率5下界合金镐秒破

  • 类型：boolean
  • 默认值：false
  • 有效值：true, false

08-20

itempickupDelay

修改被玩家丢出的物品再次被拾取的时间

  • 类型：int
  • 默认值：40
  • 建议值：40
  • 补充说明：
    • 设置为32767以禁用物品被拾取
    • 不能为负值

09-27

tickRateChangedMessage

在游戏速度改变时提示所有玩家，当游戏速度不为20gt/s时提示新玩家当前游戏速度发生改变。
启用/leavemessage 指令，允许玩家进行留言

  • 类型：boolean
  • 默认值：false
  • 有效值：true, false

10-9

easyFakePeace

允许玩家通过/fakepeace指令快速操控伪和平

  • 类型：boolean
  • 默认值：false
  • 有效值：true, false
  • 补充说明
    • 使用/setfakepeace < dimension > < pos > 设置伪和平开关的位置(开关必须为拉杆)
    • 使用/fakepeace < dimension > < boolen > 设置伪和平的状态
    • 服务器开启&开关伪和平时会自动加载伪和平开关所在区块

10-21

允许玩家通过/easycommand指令快速操控一些命令

  • 类型：boolean
  • 默认值：false
  • 有效值：true, false

12-04

tntTeleportThroughNetherPortal

允许被点燃的TNT穿过地狱门

  • 类型：boolean
  • 默认值：false
  • 有效值：true, false

12-11

disableNetherPortal

禁用地狱门

  • 类型：boolean
  • 默认值：false
  • 有效值：true, false

12-18

netherPortalCooldown

更改玩家以外实体穿过地狱门的冷却时间

  • 类型：int
  • 默认值：300
  • 建议值：300
  • 补充说明：
    • 不能为负值
